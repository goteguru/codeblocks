1. lecke – JavaScript alapmodell

Értékek, típusok, objektumok és függvények

Bevezetés

Ebben az első leckében megismerkedünk a JavaScript gondolkodásmódjának alapjaival. Nem az a cél, hogy mindent azonnal részletesen megértsünk, hanem az, hogy képet kapjunk arról, miből épül fel egy JavaScript program, és hogyan „gondolkodik” a nyelv.

A JavaScript nem varázslat: utasítások sorozata, amelyek értékekkel dolgoznak. Ezeket az értékeket típusokba sorolja, az értékeket gyakran objektumokban tárolja, és nagyon sok mindent függvényekkel old meg. Ezeket a fogalmakat most alapozzuk meg.

Mi az a programozás?

A programozás nem más, mint utasítások megfogalmazása egy számítógép számára. Ezek az utasítások pontosak, egyértelműek, és mindig ugyanúgy hajtódnak végre.

Egy program tipikusan:

  • adatokat kap (például felhasználói bevitel),
  • feldolgozza azokat,
  • majd eredményt ad vissza (például megjelenít valamit).

A JavaScript eredetileg weboldalak „életre keltésére” készült, ma viszont szervereken, mobilalkalmazásokban és sok más környezetben is használják.

Mi az a JavaScript?

A JavaScript egy dinamikus, gyengén típusos, magas szintű programozási nyelv. A lényeg kezdőként:

  • futás közben derül ki, milyen típusú adat micsoda,
  • a nyelv rugalmas,
  • sok dolgot „elnéz” a programozónak,
  • ugyanakkor összetett rendszerek építhetők vele.

Értékek és típusok

A JavaScript világában minden érték. Egy szám, egy szöveg, egy igaz/hamis állítás – mind érték. A JavaScript minden értékhez rendel egy típust, amely megmondja, hogy az adott értékkel mit lehet csinálni.

Object és function

Minden, ami nem primitive, az Object.

Mi az az Object?

Egy object kulcs–érték párok (property-k) gyűjteménye.

Ebben:

  • name és age property-k,
  • "Anna" és 30 értékek.

Egy property értéke bármilyen típus lehet.

Függvény mint property → metódus

Ha egy object egyik property-je egy függvény, akkor azt metódusnak nevezzük.

Függvények – használati szinten

A függvény egy újrahasználható művelet.

Argumentum fogalma

Az argumentum az az érték, amit átadunk a függvénynek.

Ugyanaz a függvény, különböző argumentumokkal.

Összefoglalás

Ebben a leckében megtanultuk, hogy:

  • a JavaScript program értékekkel dolgozik,
  • az értékeknek típusuk van,
  • vannak primitive és speciális primitive típusok,
  • minden összetett adat Object,
  • a függvények központi szerepet játszanak,
  • már most használunk függvényeket, még ha nem is értjük őket teljesen.

Ez az alap, amire a következő leckékben minden további tudás épülni fog.
