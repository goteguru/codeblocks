JavaScript tanfolyam

Tananyag és gyakorló feladatlapok. A feladatok oldalak gyakorlásra valók; a tananyag oldalak a magyarázatot és mintákat tartalmazzák.

Tananyagok

  • 1. lecke JavaScript alapmodell: értékek, típusok, objektumok, függvények
    Alap fogalmak, típusok, objektumok, függvényhívás.
  • 2. lecke Operátorok, típuskonverzió és vezérlési szerkezetek
    Kifejezések, összehasonlítás, truthy/falsy, if, for, while.
  • 3. lecke Függvények mélyebben
    Paraméterek, return, függvényfajták, callback szemlélet.
  • 4. lecke Fontos Object típusok
    Tömbök, objektumok, hasznos beépített típusok.
  • 5. lecke Hibakezelés, modulok, prototype
    try/catch/finally, import/export, prototype lánc.
  • 6. lecke Aszinkron JavaScript és eventek
    Promise, async/await, eseménykezelés, event loop alapok.
  • 7. lecke JavaScript mint rendszer: végrehajtási modell, closure, aszinkron minták
    Call stack, task/microtask, állapotkezelési szemlélet, másolás/mutáció.
