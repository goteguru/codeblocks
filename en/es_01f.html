Lesson 1 – Practice Exercises

Try to solve the exercises below! If you get stuck, you can ask for help from an LLM (large language model or "AI"), but if possible, try to do it on your own.

In the code, ... indicates what you need to fill in or modify. If there are complete code snippets, feel free to leave them as they are. If it's not (completely) clear what they do, that's okay; there will be more explanation later. The main point here is that you know what (and how) to write in the dotted parts.

1) Recognizing and completing types

Complete the missing values so that the variables receive the specified types.

Hint
  • undefined is created when a variable has no value or you explicitly assign undefined.
  • null is a specific "deliberately empty" value.

2) Statement or expression? (modification)

The program is incorrect because an expression is used in the wrong place as a "statement". Its result (although it is created) gets lost. Rewrite it so that the code runs and also outputs the result!

Hint

If you want to use the value of the expression later, assign it to a variable! We use the equals sign for this: variable = expression

3) Object properties

The task: in a book object, some important properties are missing. Complete it so that the output works and makes sense.

Hint

Properties are separated by commas, and after the key comes a colon: key: value.

4) Adding a method

The task: add a greet method (function property) to the person object, which prints to the console: <name> today is <year> years old!, e.g., "Anna today is 30 years old!".

Hint

You can define the method like this: greet: function () { ... }

5) Function call and arguments

The task: console.log is a function that takes arguments. The program should output a sentence based on the name and age. Correct / complete the ...TODO... part so that you get a meaningful output.

Hint

console.log can accept multiple arguments, and it prints them separated by spaces. So, just list them separated by commas.

6) Defining and using your own function

The task: define a function called describeValue that takes a value, and prints to the console the type of the value and the value itself.

Hint

Use the typeof operator: typeof value. You can also refer to the above for help.
